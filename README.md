## 📚 项目简介

`llm-self` 是我的个人 LLM 学习仓库，按照章节逐步推进，每一章对应一个小项目、一个知识点或一个实验。

主要用于记录：

- Python 基础开发
- LLM API 调用
- Prompt 设计
- 简单 Web Demo
- RAG / 文档问答 / Agent 等应用实践
